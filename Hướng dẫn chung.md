**Phương án 1: Làm thủ công**
Để AI đọc file và chỉnh sửa file đè lên.

Lợi ích: Rất linh hoạt. Cái gì cũng làm được.
Nhược điểm: Đôi khi tốn thời gian và độ chính xác mỗi lần đều có thể khác nhau.

**Phướng án 2: Lai - Hybrid**
Tạo sẵn 1 đoạn script - code - lập trình bằng Python để AI gọi và xử lý vấn đề khi cần.

Lợi ích: Độ chính xác cao hơn, chạy nhanh hơn. Đủ linh hoạt.
Nhược điểm: Vẫn cần gọi qua giao diện chat. Ko nhìn thấy kết quả trực quan

**Phương án 3: Code tool nhỏ**
